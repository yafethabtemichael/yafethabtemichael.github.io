Question 2:
Define Global Scope and Local Scope in Javascript

Answer:
Globel Scope: in which all the variables are accessible in the whole file and also can be accessed with the window object.
Local Scope: which is only accessible to the specific method. If it is declared in the a method it will be accessible only in that method.
